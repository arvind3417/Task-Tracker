### Getting started




### clone
```bash
git clone https://github.com/arvind3417/leap-assignment.git
```
```bash
cd leap-assignment
```



### Docker
```bash
docker network create leap
```

```bash
cp .env.example .env && cp .env.test.example .env.test
```

```bash
docker-compose up -d
```

```bash
docker-compose -f ./docker-compose.test.yml up -d
```
