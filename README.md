
### Technologies
1) Nestjs
2) MongoDB
3) Docker
4) Microservices


### Getting started




### clone
```bash
git clone https://github.com/arvind3417/Task-Tracker.git
```
```bash
cd Task-Tracker
```



### Docker
```bash
docker network create leap
```

```bash
cp .env.example .env && cp .env.test.example .env.test
```

```bash
docker-compose up -d
```

```bash
docker-compose -f ./docker-compose.test.yml up -d
```
